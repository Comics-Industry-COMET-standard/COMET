# The COMET Standard

**COMET** is an open metadata standard for the comics industry — a common language for
describing comics products (periodicals, graphic novels, and related items) across
publishers, distributors, and retailers.

This site is the **always-current human-readable reference**. Every field and picklist
page you see is generated directly from the standard's YAML source at build time, so it
can never drift from the canonical data.

## The three data tiers

## How to contribute

The standard is *living*. Anyone can propose a change — see
[CONTRIBUTING](https://github.com/Comics-Industry-COMET-standard/COMET/blob/main/CONTRIBUTING.md).
In short: open an issue using one of the templates, or submit a pull request. All changes
are reviewed and merged by the COMET Standards Committee.
